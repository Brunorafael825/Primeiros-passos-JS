## 🎯 Como Utilizar

### 1. **Executar Arquivos HTML Localmente**

Os arquivos `.html` podem ser abertos diretamente no navegador:

- Navegue até a pasta da aula desejada
- Clique duas vezes no arquivo `.html` ou arraste para o navegador
- Alternativa: Use um servidor local (VS Code Live Server é recomendado)

### 2. **Visualizar Scripts JavaScript**

Os arquivos `.js` contêm lógica em JavaScript:

- Abra em um editor de código (VS Code)
- Execute via Node.js: `node arquivo.js`
- Ou execute pelo console do navegador (F12)

### 3. **Estrutura Típica dos Exercícios**

A maioria dos exercícios intermediários segue este padrão:

```
ex0XX/
├── modelo.html      → Estrutura HTML
├── estilo.css       → Estilos CSS
└── script.js        → Lógica JavaScript
```

## 🛠️ Pré-requisitos

- Um navegador moderno (Chrome, Firefox, Edge, Safari)
- Um editor de código (VS Code recomendado)
- Node.js (opcional, para executar scripts .js)

## ⚡ Quick Start

1. Clone ou baixe este repositório
2. Abra a pasta no seu editor de código
3. Para visualizar: abra os arquivos `.html` no navegador
4. Para estudar: leia os arquivos `.js` e `.html` com o editor
5. Para testar: use o DevTools (F12) ou execute com Node.js
